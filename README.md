# guoming wexin 2.0

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

##目录结构

-----------

+---build 打包相关配置
+---config 打包相关配置
\---src 代码
　　+---assets 静态资源
　　|　　\---images 图片
　　+---components 组件
　　|　　\---footer 组件
　　+---filters 过滤函数
　　+---plugins 插件
　　+---router 路由配置文件
　　+---styles 样式文件
　　+---utils 工具方法
　　\---views 业务视图
　　    +---home 首页
　　    \---mine 我的
        
-----------
